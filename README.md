# [ AxeraSigConverter ]
Convert your Sigs in Style

## Features
* Convert IDA Style Sigs / Code Style Sigs
* Custom UI using ImGui Library Framework
* Lightweight Application
* Copy Converted Sigs right to your Clipboard
